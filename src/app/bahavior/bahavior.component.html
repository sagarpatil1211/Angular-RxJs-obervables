BehaviorSubject

A BehaviorSubject holds one value .

When it is subscribed it emits the value immediately

A Subject does not hold a value

// Behavior subject
        let bSubject = new BehaviorSubject(10);
        bSubject.subscribe(res=>{
            console.log('Behavior subject subs 1',res);
            })
            
        bSubject.next(20)

        bSubject.subscribe(res=>{
            console.log('Behavior subject subs 2',res);
                })
